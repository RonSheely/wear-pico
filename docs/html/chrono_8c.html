static bool _chrono_cb ( repeating_timer_t *  r)
static

The callback function for the stopwatch event.

Periodically updates the timer, calls for redraw if the primary screen is SCREEN_CHRONO. Inserts time results as follows:

  • millisecond -> second
  • second -> minute
  • minute -> hour
