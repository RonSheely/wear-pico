Splits the input string into an array of substrings, considering both the line width and '
' character.

When the line is split due to the it's width, last word will be wrapped as whole.

Parameters
buffer- input string to split
buffer_s- size of the buffer
width- maximum line width
array- array to store substrings.
array_s- size of the array
Returns
- actual number of lines in the array, -1 when improper inputs are given.

strwrap handles the string in place and affects the original string. Inserts "..." if the array size is exceeded
